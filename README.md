cf-keylayout
============
Clavier canadien français pour Mac OS X

Installation
------------
Download, unzip and from Terminal, run `./install`. Then go to System Prefs, International
(*Language and Text* for Snow users), Input Menu and check off “Canadien
Francais <version>″.

Recipes and protips
-------------------

--------               | ----------- | -----------

Keymaps
-------
* 0: Normal
* 1: Shift
* 2: Caps Lock
* 3: Alt
* 4: Alt+Shift
* 5: Alt+Caps Lock
* 6: Cmd
* 7: Ctrl
